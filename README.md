# G13
Participantes: Emiliano, Elberty.

Objetivo: Monitoração de tensão de fonte de 19V simulando tensão de placa fotovoltaica. Será feita leitura de tensão através de placa arduíno ESP32, envio de dados via wireless para roteador e em seguida transmitidos via wifi para computador-servidor.

Hardwares necessários:
1 - Fonte de tensão 19V
1 - Placa ESP32
1 - Roteador A930H
1 - Computador

Softwares necessários (a definir):
Eclipse
Apache NetBens IDE 15
Visual Studio Code
